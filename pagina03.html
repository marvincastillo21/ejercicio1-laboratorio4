Elementos de un sistema de informacion computacional

1.1 Equipo computacional

Hardware necesario para que el sistema de informacion funcione,integrado por computadoras y equipo periferico conectado a ellas.

1.2 Recurso Humano

Son las personas que interactuan con el sistema de informacion,subministran datos o utilizan el sistema para generar datos.

1.3 Fuente

Son los datos e informacion que se introducen en el sistema, es decir, son las entradas requeridas para que el sistema informatico funcione.

1.4 Programas

Software ejecutado por la computadora, el cual procesa la informacion de entrada para generar los resultados esperados.

1.5 Telecomunicaciones

Abarca el software y hardware que permite la transmision en forma electronica de texto e imagenes,entre otros.

1.5 Politicas y reglas de operacion

Aplicadas a los procedimientos y mecanismos para hacer de una aplicacion de computadora trabaje de acuerdo con estas o regir la parte funcional del proceso administrativo de la empresa.
